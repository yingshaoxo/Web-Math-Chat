# Web-Math-Chat
A simple real time math chat web app built on `parcel`, `bootstript` and `flask`.

### Demo
heroku: [https://web-math-chat.herokuapp.com](https://web-math-chat.herokuapp.com)

### Python Env
```
cd RESTful_server
sudo pip3 install -r requirements.txt
```

### Javascript Env
```
cd front-end_app
yarn
```
